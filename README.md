# Klipper LED State

**Klipper LED State** — это простой проект для создания красивой настраиваемой **LED-статусной строки для 3D-принтера**, вдохновлённая H2D, с отображением состояния принтера, текущего процесса печати, температуры стола и эффектами свечения (breath) и волны (wave).  

Проект полностью **работает «из коробки»**, без необходимости изменения конфигурации принтера. Вся обработка эффектов и управление лентой выполняется **на дешёвой плате ESP32**, что обеспечивает **низкую себестоимость реализации**. Прошивка получает статус принтера по **Wi-Fi** из локальной сети через **WebSocket**.

---

## Поддерживаемые платы

- **ESP32** (классический)  
- **ESP32-C3**  
- **ESP32-S3**  

Прошивка компилируется в **Arduino IDE** для всех этих плат.  

---

## Особенности

- Низкая себестоимость реализации: вся обработка эффектов выполняется на **ESP32**, без дополнительных контроллеров.  
- Поддержка **любых адресных светодиодных лент 5В** (WS2812/NeoPixel и совместимые).  
- Автоматическое получение статуса принтера через локальную сеть.
- Настраиваемый Data Pin, Тип Ленты и количество диодов в ленте.
- Настраиваемые **цвета и эффекты** с предпросмотром (Для удобной настройки, при выборе цвета в Web-интерфейсе, на ленте отобразится выбранный цвет на 2 секунды):
  - Breath (дыхание) - глобальный эффект
  - Wave (волна) - для статичных статусов, можно, например, включить только для паузы.
- Показывает:  
  - Текущий статус печати в двух настраиваемых цветах (плавный градиент) в процентах, начиная с левой или правой стороны или с центра.
  - Температуру стола в статусах standby и complete в виде температурной шкалы по краям ленты. 
  - Состояние принтера - отдельные цвета на standby, pause, complete и error (offline)
- Прошивка не требует **редактирования конфигурации принтера**.

## Сборка

Для работы проекта понадобится минимальный набор компонентов:  

- Плата **ESP32**, **ESP32-C3** или **ESP32-S3**  
- Любая адресная LED-лента 5В (WS2812/NeoPixel и совместимые)  
- Источник питания для платы и ленты (5В, ток зависит от длины ленты).
  Я использовал DC-DC 24в -> 5в с Aliexpress. 
- Провода для подключения  

### Схема подключения (пример для WS2812)


ESP32 → WS2812 LED Strip
GND → GND
5V → +5V ленты
GPIOXX → Data In (поддерживаются GPIO04,GPIO05,GPIO01,GPIO02, настраиваются в прошивке)
- Подключайте **GND ESP32 к GND ленты** обязательно.  
- Для стабильной работы можно добавить **конденсатор 1000µF 6.3V+** между +5В и GND ленты.  
- Для защиты данных от помех можно добавить **резистор 220–470Ω** на линии DATA.
Я обошелся без конденсаторов, все работает стабильно.

## Прошивка и настройка

### Компиляция (не обязательно, см. раздел Releases)

1. Откройте проект в **Arduino IDE**.  
2. Выберите плату:  
   - **ESP32 Dev Module** для классического ESP32  
   - **ESP32C3 Dev Module** для ESP32-C3  
   - **ESP32S3 Dev Module** для ESP32-S3  
3. Установите скорость загрузки: 115200.  
4. Компилируйте проект (`Ctrl+R` или кнопка «Verify/Compile»).  

### Загрузка прошивки

- **Готовые бинарники:**  
  Если вы не хотите собирать прошивку самостоятельно, можно скачать готовую прошивку в разделе релизов (`_full.bin`) и прошить плату через браузер с помощью сервиса:  
  [https://esptool.spacehuhn.com/](https://esptool.spacehuhn.com/)  
  - В поле адрес укажите **0x00**.
    
- **Через USB:**  
  1. Подключите ESP32 к ПК.  
  2. Нажмите «Upload» в Arduino IDE.  

- **Через OTA (по Wi-Fi, для обновления):**  
  1. Используйте `_OTA.bin` из релизов для вашей платы.  
  2. Подключитесь к плате через Web-интерфейс , выбирайте прошивку и нажимайте Update Firmware

### Настройка Wi-Fi

После прошивки **ESP32 создаст Wi-Fi точку доступа** с именем:  

Klipper_State_Led_Config
Пароль: 12345678

1. Подключитесь к этой сети с вашего телефона или компьютера.  
2. Перейдите на адрес: **192.168.4.1**  
3. В разделе **Wi-Fi** можно указать:  
   - Сеть вашего Wi-Fi и пароль для подключения ESP32 в этой сети
   - Или оставить как есть и подключаться принтером прямо к этой точке доступа 
4. В разделе Printer Settings нужно указать IP принтера и обновить страницу.
- После подключения к выбранной сети плата начнёт получать статус принтера по WebSocket.  

### Настройка ленты в Web-интерфейсе

- На странице настроек LED Config также нужно указать:  
  - **Пин подключения DATA**  
  - **Количество светодиодов**
  - **Направление статуса (слева на право или справа на лево)**
- В Color Config:
  - Глобальную яркость всей ленты
  - Цвета для различных состояний принтера
  - Эффекты Breath (амплитуду и период мерцания), Wave для конкретного статуса
  - Максимальный размер температурной шкалы стола, если она включена.
